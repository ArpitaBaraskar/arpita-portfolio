## 🏗️ Getting Started

To run the portfolio locally:

```bash
# Clone the repo
git clone https://github.com/ArpitaBaraskar/arpita-portfolio.git

# Navigate to the project directory
cd arpita-portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
